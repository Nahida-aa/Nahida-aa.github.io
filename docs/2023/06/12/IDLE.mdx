## use

- 新建文件: `Ctrl + N`
- 代码补全: `Ctrl + Space` (在 Mac 上是 `Control + Space`)
- 运行: `F5`
- 打开文档: `F1`
- 字体: Options -> Configure IDLE > Fonts/Tabs, 推荐 Consolas, size=18
- 注释: `Alt + 3` (在 Mac 上是 `Option + 3`)
- 取消注释: `Alt + 4` (在 Mac 上是 `Option + 4`)
- 缩进: `Tab`,  取消缩进: `Ctrl + [` (在 Mac 上是 `Command + [`)
