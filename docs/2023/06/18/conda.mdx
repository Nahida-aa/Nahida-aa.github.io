## install conda

Conda 支持多种安装方法

- Anaconda 是一个包含 Conda 的完整数据科学工具包，适合需要大量预装库的用户
  - 自带大量科学计算和数据分析库（如 NumPy、Pandas、Jupyter Notebook 等）
  - 文件较大（约 500MB+），安装后占用较多磁盘空间
- Miniconda 是一个轻量级的 Conda 版本，仅包含 Conda 和 Python，适合需要更灵活控制环境的用户
  - 文件较小（约 50MB）

#### use 系统包管理器安装

##### winget (Windows)

```sh
# 搜索 Miniconda
winget search miniconda
winget install Anaconda.Miniconda3
# or 
winget install Miniconda3
# 安装完成后会告诉你安装路径
# 例如 C:\Users\aa\Miniconda3 , 打开这个路径
# dir C:\Users\aa\Miniconda3 /b # 查看这个目录下的文件和文件夹
# 查看是否有  bin/ Scripts/ 文件夹
# 在环境变量中 创建 名为 CondaPath 的变量, 值为 
# C:\Users\aa\Miniconda3
# C:\Users\aa\Miniconda3\bin
# C:\Users\aa\Miniconda3\Scripts
# 然后蒋在名为 PATH 的变量中添加 %CondaPath%
```

### create env

```sh
# (optional) 列出所有可用的环境
conda env list
# (optional) 搜索可用的环境
conda search python
# 创建一个新的 Conda 环境，命名为 py38，使用 Python 3.8.16
conda create -n py38 python=3.8.16
# activate py38 环境
conda activate py38
```
