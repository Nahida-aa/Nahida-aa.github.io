```sh
# 随意创建一个 next 脚手架
# 例如:
npx heroui-cli@latest init -t app
# cd 到项目目录
cd <your-project-name>
# 安装依赖后 
npm i
# push 到 github
git push
# github 会自动识别这是一个 next 项目
# 当需要 github pages 时，只需要在 settings 中开启即可, 无需配置, 因为 github 会通过 next app 来自动配置
# 至此已经可以访问了, 不需要 配置 404 页面
```
